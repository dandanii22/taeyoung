- addEventListener을 활용한 다양한 기능 적용
-> 아코디언, 비디오, next/prev 등
- setTimeout :  비주얼 배너 롤링 기능 구현
- scrollTo : 버튼 클릭 시 위치에 맞게 스크롤
- 로그인, 회원가입 하나라도 빈칸일 경우 버튼 submit 시 disabled 기능
